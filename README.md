## Install

<code>npm i -d semantic-release <b>@dhruwlalan/semantic-release-config</b></code>

---

**.releaserc**
```json
{
  "extends": "@dhruwlalan/semantic-release-config",
}
```

*or in:*

**package.json**
```json
{
  "release": {
      "extends": "@dhruwlalan/semantic-release-config",
   }
}
```
